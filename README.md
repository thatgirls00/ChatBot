## 💻 프로젝트 소개
```
한경국립대학교 챗봇 서비스는 장학금, 취업, 행사 등 필요한 교내 공지사항을 찾기 위해 여러 행정 부서와 학과 홈페이지를 개별적으로 방문해야 하는 정보 탐색의 비효율성을 해결하고,
대학 생활에 필요한 필수 정보에 신속하고 정확하게 접근할 수 있는 환경을 구축하기 위해 제작되었습니다.
```
